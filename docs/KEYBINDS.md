# Keybindings Reference - WehttamSnaps-SwayFx

Complete reference for all keyboard shortcuts in your WehttamSnaps workstation.

## Modifier Keys

| Symbol | Key |
|--------|-----|
| `$mod` | Super (Windows key) |
| `$mod2` | Alt |
| `Shift` | Shift |
| `Ctrl` | Control |

### Window Sizing

Enter resize mode with `Super + R`:

| Action | Keybinding (in resize mode) |
|--------|----------------------------|
| Grow width | `Right` or `L` |
| Shrink width | `Left` or `H` |
| Grow height | `Up` or `K` |
| Shrink height | `Down` or `J` |
| Exit resize mode | `Return` or `Escape` |

## Custom User Bindings

Add your own keybindings in `~/.config/sway/config.d/keybindings`:

```bash
# Example custom bindings
bindsym $mod+Shift+N exec notify-send "Hello" "Custom notification"
bindsym $mod+Ctrl+L exec ~/.config/sway/scripts/lock.sh
```

## Workspace-Specific Bindings

These bindings only work in specific workspaces:

### Workspace 3 (Gaming)
- `Super + G` → Launch Steam
- `Super + Shift + G` → Toggle GameMode

### Workspace 4 (Streaming)
- `Super + O` → Launch OBS
- `Super + Shift + O` → Open audio router (qpwgraph)

### Workspace 5 (Photography)
- `Super + D` → Launch Darktable

- `Super + K` → Launch Krita

## Modifier Combinations Quick Reference

| Combination | Purpose |
|-------------|---------|
| `Super + Key` | Primary actions |
| `Super + Shift + Key` | Secondary/alternate actions |
| `Super + Ctrl + Key` | System actions |
| `Super + Alt + Key` | Advanced/mode actions |
| `Alt + Tab` | Window switching |

## Tips

1. **Learn vim-style navigation** - Use H, J, K, L instead of arrow keys for faster workflow
2. **Use the scratchpad** - Keep frequent apps like terminal or calculator in scratchpad
3. **Workspace shortcuts** - Memorize your most-used workspace numbers
4. **Customize liberally** - This is YOUR setup, modify it to fit your workflow
